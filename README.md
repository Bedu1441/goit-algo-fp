# Фінальний проєкт — Алгоритми та структури даних


## Опис проєкту

Цей репозиторій містить фінальний проєкт курсу «Алгоритми та структури даних».
Проєкт охоплює сім окремих завдань, що демонструють практичне використання структур даних, рекурсії, графових алгоритмів, роботи з деревами, жадібних підходів, динамічного програмування та методу Монте-Карло.

Кожне завдання реалізовано в окремому виконуваному Python-файлі.

# Опис виконаних завдань

## Завдання 1 — Однозв’язний список, реверс, сортування

Файл: `task_1_linked_list.py`

Реалізовано:
- структуру `Node` та клас `LinkedList`;
- реверс однозв’язного списку (in-place);
- сортування списку методом злиття (merge sort);
- об’єднання двох відсортованих списків;
- демонстрації та внутрішні тести.

## Завдання 2 — Рекурсія: фрактал «Дерево Піфагора»

Файл: `task_2_pythagoras_tree.py`

Реалізовано:
- рекурсивне побудування дерева Піфагора;
- параметризація рівня рекурсії;
- плавне градієнтне забарвлення;
- можливість збереження зображення у файл;
- масштабування та коректне відображення графіки.

## Завдання 3 — Алгоритм Дейкстри з використанням бінарної купи

Файл: `task_3_dijkstra_heap.py`

Реалізовано:
- алгоритм Дейкстри з використанням `heapq`;
- обчислення найкоротших шляхів для всіх вершин;
- відновлення шляху до цільової вершини;
- інтерактивний режим роботи;
- таблиця всіх відстаней;
- внутрішні тести правильності.

## Завдання 4 — Візуалізація бінарної купи

Файл: `task_4_heap_visualization.py`

Реалізовано:
- побудова дерева з масиву-купі;
- відображення значення та індексу елемента;
- автоматична розкладка вузлів;
- забарвлення відповідно до глибини;
- демонстрація перетворення списку у купу (`heapify`);
- внутрішні тести.

## Завдання 5 — Візуалізація обходів DFS та BFS

Файл: `task_5_dfs_bfs_visualization.py`

Реалізовано:
- побудова дерева;
- ітеративні DFS (стек) та BFS (черга);
- генерація індивідуального градієнта кольорів для кожного кроку;
- покроковий режим візуалізації;
- перевірочні тести.

## Завдання 6 — Жадібний алгоритм та динамічне програмування

Файл: `task_6_greedy_dp_food.py`

Реалізовано:
- жадібний вибір за критерієм калорійність/вартість;
- повноцінний алгоритм динамічного програмування (рюкзак 0/1);
- порівняння результатів двох підходів;
- табличний вивід;
- внутрішні тести.

## Завдання 7 — Метод Монте-Карло: кидання двох кубиків

Файл: `task_7_monte_carlo_dice.py`

Реалізовано:
- симуляція кидків кубиків;
- підрахунок частот появ сум 2–12;
- обчислення емпіричних ймовірностей;
- теоретичні ймовірності (аналітична формула);
- порівняльна таблиця з відхиленнями;
- стовпчиковий графік;
- внутрішні тести.

# Висновок щодо Завдання 7

Порівняння емпіричних та теоретичних ймовірностей показує:
- при значній кількості кидків (≥ 100 000) емпіричний розподіл наближається до аналітичного;
- сума емпіричних ймовірностей дорівнює 1 з точністю до похибки обчислень;
- найімовірніша сума (7) стабільно має найбільшу частоту;
- відхилення емпіричних значень від теоретичних є малим, що підтверджує коректність симуляції та принцип дії методу Монте-Карло.
